About Techlestial

Techlestial is an open-source technology studio — a home for developer tools published under the @techlestial npm scope.

What we build

Focused utilities for everyday development: git workflows (Gitlestial), structured logging (Loglestial), and APIs (Translatial). Each project gets a landing page, documentation, tests, and MIT license.

Rebirth standard

Every studio project ships at minimum:

  • Static landing + /docs
  • Vercel deploy with GitHub auto-deploy
  • TypeScript where applicable + Vitest
  • npm publish under @techlestial/*

Projects

  • Gitlestial — git CLI
  • Loglestial — Node logging
  • Translatial — GraphQL translation API (live)

Contributing

Fork any repo under github.com/techlestial, open a PR, and follow conventional commits. Gitlestial can generate your changelog.
